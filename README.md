# 作品集网站项目

## 项目介绍

这是一个使用 Next.js 16 和 Tailwind CSS 构建的个人作品集网站，旨在展示个人项目、AI创作作品、技能和联系方式。网站采用现代化的设计风格，支持响应式布局，在各种设备上都能提供良好的用户体验。

## 技术栈

- **前端框架**：Next.js 16
- **CSS 框架**：Tailwind CSS
- **动画库**：Framer Motion
- **状态管理**：React useState + useEffect
- **数据存储**：SQLite + Prisma ORM
- **类型系统**：TypeScript

## 项目结构

```
│   │   ├── components/     # 可复用组件
│   │   ├── projects/       # 项目展示页面
│   │   ├── works/         # AI作品展示页面
│   │   ├── about/          # 关于页面

│   │   ├── prompts/        # 指令集页面
│   │   ├── admin/          # 后台管理页面
│   │   └── page.tsx        # 首页
│   ├── lib/               # 工具库
│   └── generated/         # Prisma 生成文件
├── prisma/                # 数据库 schema
├── public/                # 静态资源
├── next.config.js         # Next.js 配置
├── package.json           # 项目依赖
└── README.md              # 项目说明
```

## 核心功能

### 1. 首页
- 英雄区展示个人信息和简介
- 精选项目展示
- 专业技能展示
- 响应式设计，适配不同设备

### 2. 项目展示页面
- 项目列表展示
- 项目搜索和排序功能
- 项目详情卡片
- 分页功能
- 无状态页面设计（当没有项目时显示友好提示）

### 3. AI作品展示页面
- AI图片作品展示（支持本地上传）
- 视频作品展示（自动提取视频首帧为封面）
- 分页功能（每页6个作品）
- 作品详情弹窗（显示创作提示词、标签等信息）
- Tab 切换图片/视频作品

### 4. 关于页面
- 个人简介和背景信息
- 技能展示
- 教育和工作经历

### 5. 联系页面
- 联系表单
- 社交媒体链接
- 个人联系方式

### 6. 指令集页面
- 提示词管理
- 分类管理
- 指令集管理
- 搜索和排序功能
- 分页功能

### 7. 后台管理页面
- 项目管理（添加、编辑、删除项目，支持本地上传图片）
- AI作品管理（添加、编辑、删除AI图片作品）
- 视频作品管理（添加、编辑、删除视频作品）
- 技能管理（添加、编辑、删除技能）
- 个人信息管理
- 联系信息管理
- 指令集管理
- 数据持久化到 SQLite 数据库

## 快速开始

### 环境要求

- Node.js 18.0 或更高版本
- npm 或 yarn

### 安装步骤

1. 克隆项目
   ```bash
   git clone https://github.com/ZhangMH-coder/portfolio.git
   cd portfolio
   ```

2. 安装依赖
   ```bash
   npm install
   ```

3. 初始化数据库
   ```bash
   npx prisma db push
   ```

4. 启动开发服务器
   ```bash
   npm run dev
   ```

5. 构建生产版本
   ```bash
   npm run build
   ```

6. 运行生产服务器
   ```bash
   npm start
   ```

## 数据库说明

项目使用 SQLite 作为数据库，首次运行需要初始化：

```bash
# 重置并初始化数据库
npx prisma db push --force-reset

# 仅初始化数据库
npx prisma db push
```

数据库文件 `prisma/dev.db` 不会上传到 GitHub，每次 clone 后需要重新初始化。

## 项目配置

### Next.js 配置

项目使用 `next.config.js` 文件进行配置，主要包括：

- 图片域名配置（允许从外部域名加载图片）

项目使用 `.env` 文件存储环境变量。

## 数据管理

项目使用 SQLite + Prisma ORM 进行数据持久化，主要存储以下数据：

- 项目数据（图片、技术栈、链接等）
- AI图片作品数据（支持本地上传 base64 存储）
- 视频作品数据（支持本地上传）
- 技能数据
- 联系信息
- 指令集数据
- 分类数据
